The goal of this GROBID module is to identify, extract and link entities in text from superconductors-related scientific literature. In particular, the goal is to extract superconductor materials and their related properties, such a Critical Temperature (Tc), Critical pressure, measurement methods.

As the others GROBID models, grobid-supercondcutors relies on machine learning and can use linear CRF (via Wapiti JNI integration) or Deep Learning model such as BiLSTM-CRF with or without ELMo (via DeLFT JNI integration).

The linking is implemented as Python library in a separate project grobid-superconductors-tools (private), which contains also other utilities.

The project is available on Github.

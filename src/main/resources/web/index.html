Which data we do collect?

  • Logs of each HTTP request of file, page and API, including IP address and browser type, OS and version (field User-agent)
  • Statistical information: location (country of origin), number of unique users, browser type, operating system type

How we do process the data?

  • Log files file and recorded in the log files of the application and retained for 7 days.
  • Statistical information are stored in an anonymised way,
  • Document and text processed by this service will not be persisted in any way,
  • No information collected here will be shared with 3rd party organisations or companies.

The goal of this GROBID module is to identify, extract and link entities in text from superconductors-related scientific literature. In particular, the goal is to extract superconductor materials and their related properties, such a Critical Temperature (Tc), Critical pressure, measurement methods.

As the others GROBID models, grobid-supercondcutors relies on machine learning and can use linear CRF (via Wapiti JNI integration) or Deep Learning model such as BiLSTM-CRF with or without ELMo (via DeLFT JNI integration).

Source code, documentation and every relevant information can be found on the GitHub repository.
